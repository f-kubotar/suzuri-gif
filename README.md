# YASURI

https://yasuri.herokuapp.com

### Contribution

```bash
$ npm install
```

Build client js

```bash
$ make
```

Start server

```bash
$ NDOE_ENV=production CLIENT_ID='YOUR CLIENT ID' CLIENT_SECRET='YOUR SECRET'  CALLBACK_URI='https://127.0.0.1:3000/callback' node server.js
```

see https://127.0.0.1:3000
